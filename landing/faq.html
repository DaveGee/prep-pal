Frequently Asked Questions

Find answers to common questions about PrepPal.

What is PrepPal?

PrepPal is an app designed to help you manage your emergency stock efficiently.

How do I download PrepPal?

You can download PrepPal from our official website or GitHub releases page.

Is PrepPal free to use?

Yes, PrepPal is completely free to use.

Can I use PrepPal offline?

Yes, PrepPal supports offline management of your emergency stock.

On which systems is PrepPal available?

PrepPal is available for Mac and Windows. Additionally, the web version can be accessed on any system with a modern browser.

Can I use PrepPal on a tablet?

Yes, the web version of PrepPal is fully compatible with tablets, allowing you to manage your emergency stock on the go.

Can the web version work offline?

Yes, the web version of PrepPal can work offline. However, the first use must be online to set up the application. After that, it will continue to function offline, and no data is stored on remote servers.

PrepPal is your companion for emergency preparedness, helping you stay ready for any situation.
